== Description ==

<a href="https://tomsneddon.org">TomS Pretty List</a> is a feature list block that help you easily to make a beautiful list. Support Font icon and Svg icon. You customize the icon size, font size , font color and background color etc.

= Features =

* Support set Font icon for the list heading and list contents separately
* Support set Svg icon(or <a href="https://developer.mozilla.org/en-US/docs/Glossary/Base64">base64</a>) for the list heading and list contents separately
* Support set links to list heading and list contents separately
* List Heading style , List Content style, and List Boxing style sections to control the List style.
* Support add custom online url of font icon (Fontawesome, Iconfont etc).

Translation ready

== Frequently Asked Questions ==

= Why I paste the svg code to the input field and auto changed to unknown strings? =

The unknown strings is <a href="https://developer.mozilla.org/en-US/docs/Glossary/Base64">base64</a>, because the wordpress core editor can not passing the Svg code to the frontend directly, for some secure reason.

= Why the font icon online url not work ? =

When you add an url to the Font Icon Online Url input field, you need to save/update the posts and reload the page, then you can add that font icon like: fa fa-icons to the icon input field that will display in real time in the editor. 
If you change the url you need to save/update the posts and empty cache and hard reload the browser to make it work.

= This plugin support shortcode ? =

No, this is a block of gutenberg.
